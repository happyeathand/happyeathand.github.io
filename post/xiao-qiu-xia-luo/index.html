小球下落

2022-09-27 算法题

题目描述
有一棵二叉树，最大深度为D，且所有叶子的深度都相同，。所有节点从上到下从左到右编号为1，2，3，。。。2D-1。在节点1处放一个小球，他会往下落。每个内节点上都有一个开关，初始全为闭合。每次小球落到一个开关上时，它的状态都会改变。当小球到达一个内节点时，如果开关关闭时往左走，否则往右走。直到叶子节点。

输入描述
第一行n，代表接下来有n个测试数据

接下来n行，每行对应深度D，小球编号m（代表第m个落下的小球）

输出描述
第m个小球落到的结点编号

样例输入
4
4 2
3 4
10 1
2 2
样例输出
12
7
512
3

#include<stdio.h>

int a[1000];

int main()
{
	int n;
	scanf("%d",&n);
	while(n--)
	{
		int d,m;
		scanf("%d%d",&d,&m);
		for(int i=0;i<1000;i++) a[i]=0;
		int index=1;
		for(int i=0;i<m;i++)
		{
			for(int j=1;j<d;j++)
			{
				if(a[index]==0)
				{
					a[index]=1;
					index=2*index;
				}
				else
				{
					a[index]=0;
					index=2*index+1;
				}
			}
			if(i==(m-1)) printf("%d\n",index);
			index=1;
		}
	}
	return 0;
}

这道题之前校赛是做过的，用数组模拟二叉树然后模拟就ok了，但是有个问题要注意下，第一次写完之后结果刚好是正确结果的2倍，为啥呢，一看原来是因为循环里的层数，就是for循环里二叉树深度的初始值j是从0开始的，然而应该直接从1开始就对了，因为都是从第一层放入的嘛。
