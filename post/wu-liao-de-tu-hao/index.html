无聊的土豪

2022-10-02 算法题

题目描述
经过繁忙的筹备，“图灵杯”NEUQ-ACM编程大赛终于开始了！比赛像熊熊燃烧的火焰一样进行，吐铧感受到了成就感。这时，吐锢突然想玩个游戏。他首先在脑海中思考一个随机数，然后将这个随机数的每个数字相加，形成一个新数字。如果这个数字大于或等于10，那么Tuhao将每个数字相加，再次做一个新数字，直到他得到一个小于10的数字。如果原来的随机数可以除以这个数字，那么打印“YES！”。否则，请打印“否！” 。虽然你觉得吐昊很无聊，但你还是想验证他的答案。

输入描述
有多个输入数字n（0<n<10^18），n是随机数，n=0是输入的末尾。

输出描述
打印“是！“或者”不！“，每个输出需要一条线。

样例输入
1000
1001
0
样例输出
YES!
NO!

#include<stdio.h>
int main()
{
	long long int n;
	scanf("%lld",&n);
	while(n!=0)
	{
		long long int nn,sum=0;
		nn=n;
		while(nn>=10)
		{
			while(nn!=0)
			{
				sum+=nn%10;
				nn/=10;
			}
			nn=sum;
		}
		if(n%nn==0) printf("YES!\n");
		else printf("NO!\n");
		scanf("%lld",&n);
	}
}

很简单的一道题啊，当时写的时候好像写了三四十分钟，其实后来写好像是5分钟解决。被题目的n的取值范围唬住了，10^18次方，这怎么看long long都会爆掉吧，然而long long是9*10^19,刚好爆不了
