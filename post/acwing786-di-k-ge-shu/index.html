AcWing786. 第k个数

2022-04-29 算法基础课学习记录

给定一个长度为 n 的整数数列，以及一个整数 k，请用快速选择算法求出数列从小到大排序后的第 k 个数。

输入格式
第一行包含两个整数 n 和 k。

第二行包含 n 个整数（所有整数均在 1∼109 范围内），表示整数数列。

输出格式
输出一个整数，表示数列的第 k 小数。

数据范围
1≤n≤100000,
1≤k≤n
输入样例：
5 3
2 4 1 5 3
输出样例：
3
解

#include<bits/stdc++.h>
using namespace std;

int a[100005];

int quick_sort(int l,int r,int k)
{
	if(l==r)
	return a[l];
	int i=l-1,j=r+1,x=a[l];
	while(i<j)
	{
		do i++;while(a[i]<x);
		do j--;while(a[j]>x);
		if(i<j)swap(a[i],a[j]);
	}
	
	if((j-i+1)>k) return quick_sort(l,j,k);
	else return quick_sort(j+1,r,k);
}

int main()
{
	int i,n,k;
	scanf("%d%d",&n,&k);
	for(i=0;i<n;i++)
	{
		cin>>a[i];
	}
	cout<< quick_sort(0,n-1,k);
	return 0;
}

快速选择时间复杂度是O(n)的，快排是O(nlogn)，思想是一样的，都是采用分治的思想和双指针向中间扫描，不过快排是要把每个小问题都解决，而快速选择只需要解决一个小问题，把细分的问题再不断细分，解决到最小的问题就结束了，不用把整个数列进行排序，只要把它所在的那半边排好就ok了
