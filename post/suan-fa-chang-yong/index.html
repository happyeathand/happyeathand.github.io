库函数的问题

  1. bool类型变量要定义头文件<stdbool.h>,如果自己的话不如定义int型的函数。如果是题目要求就没办法了
  2. 代码结尾一定要记得加return 0。
  3. Int型最大能存21亿多点，准确来说是-2147483647~+2147483647
    而long long int可以存-9223372036854775808～+9223372036854775807，而且长长整型的输入和输出前面都要加一个ll，譬如scanf(“%lld”,&a),printf(“%lld”,a)。
    4． 输出的补齐，补足两位用0补齐这么写：printf(“%02d”,a)
  4. double类型的输入scanf(“%lf”),输出可以是%f也可以是%lf，float应该就都是%f了。
  5. 转换绝对值时浮点型和整型的函数不一样，整型是abs（），浮点型是fabs（）。
  6. 在用“%c”格式输入字符时，空格字符和转义字符都作为有效字符输入。scnaf("%c%c%c",&c1,&c2,&c3);

    数论

  7. 计算最大公约数的函数
int gcd(int a,int b)//计算的是两个参数的最大公约数
{
    if(b==0)return a;
    else return gcd(b,a%b);
}

最后的返回值就是最大公约数了，叫做欧几里得算法（辗转相除法）

4.分解质因数
这个意思是
根据算术基本定理又称唯一分解定理，对于任何一个合数， 我们都可以用几个质数的幂的乘积来表示。
例如
12=2*2*3
20=2*2*5
30=2*3*5
接下来我们利用这个公式分解质因数。
设一个质数为p.如果n%p == 0，那么p就是n的一个质因数，接下来就是求p的指数，我们让n = n/p, 这样就从n中剔除了一个p,接着重复上述两步，直到n%p ！= 0(其实就是不断循环)

  1. 定义变量时的初值
  2. While里的递增变量记得赋值
