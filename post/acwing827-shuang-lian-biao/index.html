AcWing827. 双链表

2022-08-10 算法基础课学习记录

实现一个双链表，双链表初始为空，支持 5 种操作：

在最左侧插入一个数；
在最右侧插入一个数；
将第 k 个插入的数删除；
在第 k 个插入的数左侧插入一个数；
在第 k 个插入的数右侧插入一个数
现在要对该链表进行 M 次操作，进行完所有操作后，从左到右输出整个链表。

注意:题目中第 k 个插入的数并不是指当前链表的第 k 个数。例如操作过程中一共插入了 n 个数，则按照插入的时间顺序，这 n 个数依次为：第 1 个插入的数，第 2 个插入的数，…第 n 个插入的数。

输入格式
第一行包含整数 M，表示操作次数。

接下来 M 行，每行包含一个操作命令，操作命令可能为以下几种：

L x，表示在链表的最左端插入数 x。
R x，表示在链表的最右端插入数 x。
D k，表示将第 k 个插入的数删除。
IL k x，表示在第 k 个插入的数左侧插入一个数。
IR k x，表示在第 k 个插入的数右侧插入一个数。
输出格式
共一行，将整个链表从左到右输出。

数据范围
1≤M≤100000
所有操作保证合法。

输入样例：

10
R 7
D 1
L 3
IL 2 10
D 3
IL 2 7
L 8
R 9
IL 4 7
IR 2 2

输出样例：
8 7 7 3 2 9

题解

双链表的操作和单链表大同小异，不同的是多了一个tail节点，不知道有啥用，应该是为了方便头尾元素的插入吧，还有就是方便最后的链表遍历？（大概）。然后需要注意的就是head和tail仅作为标志，不存值，遍历也是不包括他们两个的。

#include<iostream>

using namespace std;

const int N = 1e5 + 10;

int m;
int e[N], l[N], r[N];//e[N]表示链表中元素的值，l[N]左指针指向第n个元素左边的元素的位次，r[n]同理
int idx;//第几个插入的元素
void init()//初始化链表
{
	r[0]=1;//链表的头和尾都先定义好了，head是0，tail是1
	l[1]=0;
	idx=2;//idx从2开始
}
void add(int k,int x)//这里的函数是向第k个插入的元素右面插入一个新的值x，如果是向左插入，参数改成l[k]就ok了
{
	e[idx]=x;
	l[idx]=k;
	r[idx]=r[k];
	l[r[k]]=idx;
	r[k]=idx;
	idx++;
}
void remove(int k)
{
	r[l[k]]=r[k];
	l[r[k]]=l[k];
}


int main()
{//因为c语言判断字符太麻烦所以直接找了个c++的题解copy一下主函数
	 ios::sync_with_stdio(false);
    cin >> m;

    init();

    while(m--)
    {
        string op;
        cin >> op;
        int k, x;
        if(op=="R")
        {
            cin >> x;
            add(l[1], x);//向链表最右端添加一个数，即把这个数插入在位次为1的元素的左边
        }
        else if(op=="L")
        {
            cin >> x;
            add(0, x);//和上边一个道理，向最左端插入一个数，就是把这个数插入到位次为0的元素右边
        }
        else if(op=="D")
        {
            cin >> k;
            remove(k + 1);//为什么是k+1呢，因为位次是从0开始计算的，然而init里面已经初始化了head和tail，所以已经被占了俩个，然而题目说的是第k个插入的数，又要从1开始计算，所以这里写k+1就合理了
        }
        else if(op=="IL")
        {
            cin >> k >> x;
            add(l[k + 1], x);
        }
        else
        {
            cin >> k >> x;
            add(k + 1, x);
        }    
    }
    for(int i=r[0];i!=1;i=r[i])
    printf("%d ",e[i]);
}
