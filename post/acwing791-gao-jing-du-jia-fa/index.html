AcWing791. 高精度加法

2022-06-21 算法基础课学习记录

给定两个正整数（不含前导 0），计算它们的和。

输入格式
共两行，每行包含一个整数。

输出格式
共一行，包含所求的和。

数据范围
1≤整数长度≤100000
输入样例：
12
23
输出样例：
35

题解

#include<stdio.h>
#include<string.h>

char op1[100005],op2[100005];//两个整数的输入
int a[100005],b[100005],c[100005];//字符串转换成整型用数组存储，结果也用数组输出

int main()
{
	int i,j;
	gets(op1);
	for(j=0,i=strlen(op1)-1;i>=0;i--,j++)//把加数倒序放入数组中，即个位在数组的第0位
	{
		a[j]=op1[i]-'0';
	}
	gets(op2);
	for(j=0,i=strlen(op2)-1;i>=0;i--,j++)//同理
	{
		b[j]=op2[i]-'0';
	}
	
	int len=strlen(op1)>strlen(op2)?strlen(op1):strlen(op2);//len等于较大的位数，确保把数字加完
	
	for(i=0;i<=len;i++)
	{
		if((a[i]+b[i])>=10)//大于10进位就ok
		{
			c[i]=a[i]+b[i]-10;
			a[i+1]++;//第一次写成了c[i+1]++，显然错，这里写a或b都是可以的
		}
		else
		{
			c[i]=a[i]+b[i];
		}
	}
	
	if(c[len]==0)//判断当两个加数位数相同且最高位相加是否＞10，如果不＞10，那么最后是不进位的，那么c的最高位就不会＝0
	{
		i=len-1;
	}
	else
	{
		i=len;
	}
	
	for( ;i>=0;i--)//记得做结果的数组进行倒序输出
	{
		printf("%d",c[i]);
	}
	
	return 0;
}

这道题算是一个纯模拟题，但是第一次写时没有把加数倒序存储，这就导致了如果最高位相加＞10的话结果数组还要多留一位，且要在计算前提前留，如果倒序的话就在结果后判断，额(⊙﹏⊙)，好像也差不多，都是加个判断就行了，只是顺序的话写到最后才发现要留一位。
