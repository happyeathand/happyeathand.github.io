硬币问题

2022-10-02 算法题

题目描述
现在有1元，5元，10元，50元，100元，500元的硬币各C1,C5,C10,C50,C100,C500枚。现在要用这些硬币来支付A元，最少需要多少枚硬币？

输入描述
多组数据输入，每组有两行，第一行包括一个数A（0<A<109），第二行包括包括C1,C5,C10,C50,C100,C500(0≤C1,C5,C10,C50,C100,C500<109)。

输出描述
每组数据输出一个数，即最少需要硬币的枚数

样例输入
620
3 2 1 3 0 2
样例输出
6

#include<stdio.h>

int main()

{
	int i,n,a[6],count=0;
	scanf("%d",&n);
	for(i=0;i<6;i++) scanf("%d",&a[i]);//个数 
	int b[6]={1,5,10,50,100,500};//大小 
	
	for(i=5;i>=0;i--)
	{
		while(n>0)
		{
			if(n/b[i]>0&&a[i]!=0) 
			{	
				if(a[i]>=n/a[i])
				{
					count++;
					n=n-b[i];
					a[i]--;
				}
				else
				{
					count++;
					n=n-b[i];
					a[i]--;
				 } 
			}
			else break;
		}
	}
	printf("%d",count);
}
