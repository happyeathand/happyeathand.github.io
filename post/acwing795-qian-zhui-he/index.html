AcWing795. 前缀和

2022-06-21 算法基础课学习记录

输入一个长度为 n 的整数序列。

接下来再输入 m 个询问，每个询问输入一对 l,r。

对于每个询问，输出原序列中从第 l 个数到第 r 个数的和。

输入格式
第一行包含两个整数 n 和 m。

第二行包含 n 个整数，表示整数数列。

接下来 m 行，每行包含两个整数 l 和 r，表示一个询问的区间范围。

输出格式
共 m 行，每行输出一个询问的结果。

数据范围
1≤l≤r≤n,
1≤n,m≤100000,
−1000≤数列中元素的值≤1000
输入样例：
5 3
2 1 3 6 4
1 2
1 3
2 4
输出样例：
3
6
10

题解

#include<stdio.h>

int s[100005];//数组s类似于数列里的前n项和

int main()
{
	int i,n,m,a;
	scanf("%d%d",&n,&m);
	for(i=1;i<=n;i++)
	{
		scanf("%d",&a);
		s[i]=s[i-1]+a;
	}
	
	int l,r;
	for(i=0;i<m;i++)
	{
		scanf("%d%d",&l,&r);
		printf("%d\n",s[r]-s[l-1]);
	}
	
	return 0;
}

暴力写法直接超时，利用前缀和的思想，先把所有的s[n]算出来，最多算100000次，然后后面的l与r之间的和直接用s[r]-s[l-1]就ok了
