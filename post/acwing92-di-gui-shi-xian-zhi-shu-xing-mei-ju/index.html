ACwing92. 递归实现指数型枚举

2022-03-05 算法题

从 1∼n1∼n 这 nn 个整数中随机选取任意多个，输出所有可能的选择方案。

输入格式

输入一个整数 nn。

输出格式

每行输出一种方案。

同一行内的数必须升序排列，相邻两个数用恰好 11 个空格隔开。

对于没有选任何数的方案，输出空行。

本题有自定义校验器（SPJ），各行（不同方案）之间的顺序任意。

数据范围

1≤n≤151≤n≤15

输入样例：

3

输出样例：

3
2
2 3
1
1 3
1 2
1 2 3

题解

#include<stdio.h>
int n,x;
int state[100];
void dfs(int x)
{
	int i;
	if(x>n)
	{
		for(i=1;i<=n;i++)
		{
			if(state[i]==1)
			printf("%d ",i);
		}
		printf("\n");
		return;
	}
	
	state[x]=1;
	dfs(x+1);
	state[x]=0;
	
	state[x]=0;
	dfs(x+1);
	state[x]=0;
}
int main()
{
	scanf("%d",&n);
	dfs(1);
}

思路：

对于1~n，按顺序挨个做判断，可以选择选或不选，每做出一次选择后，dfs（x+1），进行下一个位置的选择，直到x>n即判断完毕，得到了一个答案，然后再退出本次递归，上层递归函数继续顺序往下执行，直到最后最外层代码执行完毕，退出。

与94题不同的是，这道题的dfs在输出完毕后需要加一个return，这是因为94题刚好i>n会退出函数，所以一般来讲的话还是要加个return比较好。
