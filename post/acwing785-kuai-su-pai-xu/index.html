AcWing785. 快速排序

2022-04-28 算法基础课学习记录

给定你一个长度为 n 的整数数列。

请你使用快速排序对这个数列按照从小到大进行排序。

并将排好序的数列按顺序输出。

输入格式

输入共两行，第一行包含整数 n。

第二行包含 n 个整数（所有整数均在 1∼109 范围内），表示整个数列。

输出格式

输出共一行，包含 n 个整数，表示排好序的数列。

数据范围

1≤n≤100000

输入样例：

5
3 1 2 4 5

输出样例：

1 2 3 4 5

解

#include<bits/stdc++.h>
using namespace std;

int a[100005];

int main()
{
	int n,i;
	cin >> n;
	for(i=0;i<n;i++)
	{
		cin >> a[i];
	}
	sort(a,a+n);
	for(i=0;i<n;i++)
	{
		cout<<a[i]<<" ";
	}
}

以后尝试用c++写题了，这题直接sort就可ac

#include<bits/stdc++.h>
using namespace std;

const int N = 1e6+10;

int a[N];

void quick_sort(int a[],int l,int r)
{
    if(l>=r)
	return;
	int i,j,x;
	x=a[(l+r)/2];//取中点，取边界值会超时
	i=l-1;
	j=r+1;
	while(i<j)
	{
		do i++;while(a[i]<x);
		do j--;while(a[j]>x);
		if(i<j) swap(a[i],a[j]);
	}
	
	quick_sort(a,l,j);
	quick_sort(a,j+1,r);
}

int main()
{
	int n,i;
	cin >> n;
	for(i=0;i<n;i++)
	{
		cin >> a[i];
	}
	quick_sort(a,0,n-1);
	for(i=0;i<n;i++)
	{
		cout<<a[i]<<" ";
	}
}

看视频题解后的解法，采用分治的思想，把一个大问题分成一个个小问题，然后再用递归去解决每一个小问题，这道题是利用两个指针（i，j）分别从头和尾（l，r）向中间靠拢，每靠拢一次，就和值x进行一次比较（x的值在后面说明），若a[i]>x则i停止扫描，若a[j]<x则j停止扫描，ij都停止扫描后再交换值继续扫描，这样结束后大于x的值都在右边，小于x的值都在左边。随后递归改变边界值的位置继续扫描，直到两个指针位置重合，然后以同样的方式再扫另一半，程序结束。
x的值本来是取的左边界，但是超时了，后来看评论区有人说是如果输入已经是排好序的输入了，那么费的时间会很久，看来是这样的

其实大概就是如果是个递增的数列，那么以边界值做x的话，递归处理右半部分的次数会相当多，时间复杂度也就上去了，但若是取中值（相当于一个随机值，只要不是边界值就好hh）的话，是会比取边界值快不少（至少在有序数列里是这样）。
